# EasyAnyLink

中文文档 | [English](./README.md)

EasyAnyLink 是一个由两部分组成的覆盖网（overlay networking）系统，用于将分散的私有网络统一为一个可达的空间。它包含一个面向公网上的 Server（服务器）以及可插拔的 Agent（代理），Agent 承担两种角色：Client（客户端）和 Gateway（网关）。

## 组件

- Server：面向互联网的协调与数据中继节点。负责代理注册、会话编排、鉴权，并在代理之间中继流量。
- Agent：
  - Client：在本地创建 TUN 接口，并安装默认路由（或可选的分流路由），将流量发送到覆盖网络中。
  - Gateway：通过覆盖网络接收来自 client 的 TUN 数据包，并将其转发到本地网络或互联网，类似于 VPN 的出站/代理节点。

数据路径

client ↔ server ↔ gateway

功能亮点

- 从任意位置安全访问位于 gateway 私有网络中的资源。
- 使用 gateway 作为对外出口访问公共互联网，支持代理式浏览或服务访问。

设计目标

- 简单部署：一个公网服务器，多个代理节点。
- 基于第 3 层（Layer-3）的透明度，使用 TUN 接口。
- 安全优先的架构，支持相互认证和加密通道。
- 灵活的路由策略（默认路由或分流）和多租户准备。

## #实现细节

### Server

Server 是 EasyAnyLink 架构的核心组件，负责：

- 代理注册与管理
- 会话编排与状态管理
- 代理的认证与授权
- 在客户端与网关之间中继流量
- 网络活动的监控与日志记录

### Agent

Agent 是与 Server 与本地网络交互的关键组件，具有两个主要角色：

- Client：在本地创建 TUN 接口，并安装默认路由（或可选的分流路由），将流量引入覆盖网络。
- Gateway：通过覆盖网络接收来自 client 的 TUN 数据包，并将其转发到本地网络或互联网，起到类似 VPN 出口/代理 的作用。
